Client and Server for Manage gateways - master devices project

Vue.js + Express + MongoDB

Run npm run install for all package.json files

Installation:
 - Inside the /server folder, you can find all server logic, to start the server api run: npm run dev. 
   You can also find the public folder for client side
 - Inside /client folder can find all Vue.js code for client side 
 - In the MongoDB Database create database "gateways" and 2 collections: "gateways" and "peripherals"
   Import data from files gateways.json and peripherals.json, file indentitycounters.json contains the UID counter in peripherals
